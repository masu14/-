
実行ファイルはbin内のmersennneDraft.exeを実行してください
ゲームの動作紹介動画はmovie.mp4を実行してください
ソースコードはScripts内のファイルを実行してください

ゲームの遊び方
A,Dキー　	  横移動
Wキー　 	  ジャンプ
Sキー	  すり抜け床の通過
Enterキー	　ショット
Shiftキー　 ショット切り替え

制作環境
OS:Windows11
CPU:Intel Core i5-10400F
GPU:NVIDIA GeForse RTX 2060
使用言語:C#
使用ツール:Unity ver2021.3.9f1, visual studio 2022, Git for Windows, GitHub
プラグイン:UniRx
